Python コマンドリファレンス

1. 基本確認

インストールされているPythonのバージョンを確認します。

python --version

2. 仮想環境 (venv) の操作

プロジェクトごとに独立した環境を作るための標準機能です。

仮想環境の作成
python -m venv .venv
仮想環境の有効化 (Activate)

Windows

.venv\Scripts\activate

macOS / Linux

source .venv/bin/activate
環境の無効化 (Deactivate)
deactivate

3. パッケージ管理 (pip)

ライブラリのインストールや管理を行います。

操作 コマンド
ライブラリのインストール pip install パッケージ名
特定のバージョンを指定 pip install パッケージ名==1.2.3
インストール済み一覧 pip list
ライブラリの削除 pip uninstall パッケージ名

4. requirements.txt の活用

チーム開発やデプロイ時に環境を再現するために使用します。

現在の環境構成を書き出す
pip freeze > requirements.txt
ファイルから一括インストール
pip install -r requirements.txt

5. スクリプトの実行

python main.py
Tips: Windowsで複数のPythonがインストールされている場合、pythonの代わりにpyコマンド（Pythonランチャー）を使用すると便利な場合があります。
例: py -3.10 -m venv .venv
