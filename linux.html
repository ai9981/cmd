Linuxコマンド実用100選

サーバー管理や開発業務で必須となる、主要Linuxコマンドとその用法を完全網羅。

1. ファイル・ディレクトリ操作

コマンド解説
lsファイル・ディレクトリの一覧を表示
ls -la隠しファイルを含む詳細情報を表示
cd [dir]ディレクトリを移動
pwd現在のカレントディレクトリを表示
mkdir [dir]新しいディレクトリを作成
mkdir -p [path]中間ディレクトリも含めて再帰的に作成
rm [file]ファイルを削除
rm -rf [dir]ディレクトリを中身ごと強制削除（注意！）
cp [src] [dst]ファイルをコピー
cp -r [src] [dst]ディレクトリを再帰的にコピー
mv [old] [new]ファイルの移動、または名前変更
touch [file]空ファイル作成、またはタイムスタンプ更新
ln -s [src] [link]シンボリックリンク（ショートカット）を作成
treeディレクトリ構造をツリー形式で表示
file [file]ファイルの種類（バイナリ、テキスト等）を確認

2. テキスト処理・閲覧

コマンド解説
cat [file]ファイルの内容を全て表示
less [file]大きなファイルをスクロールして閲覧
head -n [num]ファイルの先頭数行を表示
tail -n [num]ファイルの末尾数行を表示
tail -f [file]ログファイルなどの更新をリアルタイム監視
grep "[word]"特定の文字列を検索
grep -r "[word]" [dir]ディレクトリ内を再帰的に文字列検索
sed 's/old/new/g'文字列の置換処理
awk '{print $1}'テキストの列（カラム）を抽出して処理
cut -d',' -f1区切り文字を指定して特定のフィールドを抽出
sort行をアルファベット・数値順に並び替え
uniq重複する行を削除（隣接行のみ）
wc -l [file]ファイルの行数をカウント
diff [file1] [file2]2つのファイルの差分を表示
echo "text"標準出力に文字列を表示

3. システム情報・管理

コマンド解説
uname -aOSカーネルやシステム情報を表示
hostnamectlホスト名の確認・設定
uptimeシステムの稼働時間と負荷（Load Average）を表示
free -mメモリの空き容量をMB単位で表示
date現在のシステム日時を表示・設定
calカレンダーを表示
dmesgカーネルの出力メッセージ（ログ）を表示
lscpuCPUのアーキテクチャ詳細を表示
lsusb / lspci接続されているUSB/PCIデバイスを表示
vmstatリソースの使用状況を一覧表示

4. プロセス管理

コマンド解説
ps aux実行中の全プロセスを詳細表示
topプロセスのリアルタイム監視（CPU/メモリ順）
htopより視認性の高いプロセス監視（要インストール）
kill [PID]特定のプロセスを終了
kill -9 [PID]プロセスを強制終了
pkill [name]プロセス名で指定して一括終了
jobsバックグラウンドで実行中のジョブを表示
bg / fgジョブをバックグラウンド/フォアグラウンドに切替
nohup [cmd] &ログアウト後もコマンドを実行継続
systemctl status [svc]サービスの稼働状態を確認

5. ネットワーク関連

コマンド解説
ip addrIPアドレスなどのネットワーク設定を表示
ping [host]接続先端との疎通確認
curl [URL]URLを叩いてレスポンスを取得（HTTP/HTTPS）
wget [URL]ファイルをダウンロード
ssh [user]@[host]リモートサーバーに安全にログイン
scp [file] [dst]SSH経由でファイルを転送
rsync -av [src] [dst]効率的なファイル同期・バックアップ
ss -tlpnリッスンしているポート番号を表示
netstat -anネットワーク接続状態の一覧表示
dig [domain]DNS情報を照会
nslookup [domain]ドメイン名からIPを引く（またはその逆）
traceroute [host]通信経路を確認

6. 権限・ユーザー管理

  • userdel -r [name]
    • コマンド解説
    sudo [cmd]管理者権限（root）でコマンドを実行
    chmod [mode] [file]ファイルやディレクトリの権限を変更
    chown [user]:[group]所有者とグループを変更
    su - [user]ユーザーを切り替える（- で環境変数を引継ぐ）
    whoami現在のログインユーザー名を表示
    idユーザーIDや所属グループを表示
    passwdパスワードを変更
    useradd [name]新規ユーザーを作成
    ユーザーを削除（ホームディレクトリ込み）
    groups所属しているグループを表示

    7. 圧縮・解凍・アーカイブ

    コマンド解説
    tar -cvf [file.tar] [dir]アーカイブ（固める）を作成
    tar -xvf [file.tar]アーカイブを展開
    tar -zcvf [file.tar.gz]gzip形式で圧縮してアーカイブ
    tar -zxvf [file.tar.gz]gzip形式のアーカイブを解凍・展開
    gzip [file]ファイルをgzip形式で圧縮
    gunzip [file.gz]gzipファイルを解凍
    zip -r [file.zip] [dir]ZIP形式で圧縮
    unzip [file.zip]ZIPファイルを解凍

    8. 検索・場所特定

    コマンド解説
    find [dir] -name "[pattern]"ファイル名でファイルを検索
    find [dir] -type f -mtime -7最近7日間に更新されたファイルを検索
    locate [file]DBから高速にファイルを検索（updatedbが必要）
    which [cmd]実行コマンドのパス（場所）を表示
    whereis [cmd]バイナリ、ソース、マニュアルの場所を表示
    type [cmd]コマンドの種類（別名、関数、外部）を表示
    grep -l "[word]" *指定文字列を含むファイル名のみ表示
    history過去に実行したコマンドの履歴を表示

    9. ディスク・ストレージ

    コマンド解説
    df -hディスクの使用量（人間が読みやすい形式）を表示
    du -sh [dir]特定のディレクトリの使用量を集計
    lsblkブロックデバイス（HDD/SSD/パーティション）を表示
    mountデバイスのマウント状態を表示・実行
    umount [path]マウントを解除
    fdisk -lパーティションテーブルを表示

    10. その他・便利コマンド

    コマンド解説
    alias [name]='[cmd]'コマンドの別名（エイリアス）を設定
    export [VAR]=[val]環境変数を設定
    source [file]設定ファイル（.bashrcなど）を即座に反映
    watch -n [sec] [cmd]コマンドを一定間隔で繰り返し実行
    crontab -e定期実行タスク（スケジュール）を編集
    shutdown -h now即座にシステムを終了
    rebootシステムを再起動
    clearターミナルの表示をクリア
    man [cmd]コマンドのマニュアルを表示
    exitセッションを終了
